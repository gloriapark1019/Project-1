Basic Tags

Basic HTML tags

Tag Name Description
<!DOCTYPE HTML> HTML Doctype Declaration The<!DOCTYP> declaration must be the very first thing in your HTML document, before the <html> tag.
<html> HTML The <html> tag tells the browser that this is an HTML document.
<title> Title Required in all HTML documents and it defines the title of the document displayed in browser toolbar.
<div> div The <div>tag defines a division or a section in an HTML document.
<p> Paragraph Tag The<p>tag defines a paragraph.
<span> Span Tag The <span>tag is used to group inline-elements in a document. Provides a way to add a hook to a part of a text or a part of a document.
<ul> Unordered List The <ul> tag defines an unordered (bulleted) list. Enclose each item with <li>
<ol> Ordered List The <ol> tag defines numbered list. Enclose each item with <li>

Table

HTML Tags for Making a Table

Tag Name Description
<table> Table The <table> defines an HTML table.
<thead> Table Head Used to group header content in an HTML table.
<tbody> Table Body Used to group the body content in an HTML table.
<tfoot> Table Foot Used to group footer content in an HTML table.
<tr> Table Row The <tr> defines a row in an HTML table, contains one or more <td> or <th> elements.
<th> Table Head The <th> defines a header cell in an HTML table. Bold and center-aligned by default.
<td> Table Data The <td> defines a standard cell in an HTML table. Left-aligned by default.
